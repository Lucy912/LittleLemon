# Training Content Creator - Claude Skill

**Purpose:** Create comprehensive, audience-focused training materials for online courses and workshops
**Domain:** Educational content creation, instructional design, video training
**Version:** 1.0

---

## Philosophy

### Core Principles

1. **Audience-First Design**
   - Understand pain points before creating solutions
   - Speak to emotional state, not just information needs
   - Anticipate and overcome objections proactively

2. **Research-Driven Decisions**
   - Base content structure on proven formats (YouTube retention data, learning science)
   - Validate assumptions with market research
   - Use specific metrics (6-9 min video length, 60%+ retention, pattern interrupts every 2-3 min)

3. **Value-to-Friction Ratio**
   - Maximize immediate, actionable value
   - Minimize barriers to implementation
   - Deliver quick wins that build confidence

4. **Iterative Perfectionism**
   - Ship "good enough" fast
   - Improve systematically based on feedback
   - Prioritize volume and iteration over initial perfection

5. **Practical Over Theoretical**
   - 70% hands-on practice, 30% explanation
   - Every concept demonstrated with real examples
   - Participants create their own solutions during training

## Methodology: The 8-Step Training Creation Process

### Step 1: Define Training Structure
**Goal:** Create comprehensive curriculum framework before diving into details

**Actions:**
1. Identify core learning objectives
2. Break into logical modules (8-12 for full course)
3. Sequence modules for progressive skill building
4. Define deliverables for each module
5. Estimate time requirements (video length, homework, practice)

**Output:** Master training structure document with module breakdown

**Key Questions:**
- What should learners be able to DO after completing this?
- What's the logical progression from beginner to proficient?
- What's the minimum viable curriculum (MVP)?

---

### Step 2: Research Best Practices
**Goal:** Inform content strategy with proven approaches

**Research Areas:**
- Video format best practices (length, structure, engagement)
- Top-performing content in your domain (what works?)
- Audience preferences and consumption patterns
- Competitor analysis (gaps and opportunities)
- Tools and delivery platforms

**Actions:**
1. Search for top YouTube videos in topic area
2. Analyze what makes them successful (retention, comments, views)
3. Research educational video best practices
4. Document key findings and benchmarks

**Output:** Research findings document with specific, actionable insights

**Key Metrics to Discover:**
- Optimal video length
- Retention patterns
- Engagement techniques
- Format preferences
- Pricing benchmarks (if monetized training)

---

### Step 3: Audience Analysis
**Goal:** Deep understanding of who you're serving and what they need

**Segments to Define:**
- Demographics (age, role, industry, income)
- Current state (skills, knowledge, tools)
- Pain points (fears, frustrations, obstacles)
- Goals (desired outcomes, motivations)
- Objections (reasons they might not engage or succeed)

**Actions:**
1. Create detailed personas for each audience segment
2. Map emotional journey (before → during → after training)
3. Identify universal needs vs. segment-specific needs
4. Prioritize audience segments (primary, secondary, tertiary)

**Output:** Audience analysis document with personas and needs mapping

**Critical Insight:** You're not teaching content, you're transforming people from one state to another.

---

### Step 4: Content Outlining
**Goal:** Structure individual session with precise timing and flow

**Framework:**
1. **Hook (0:00-0:15):** Grab attention, create curiosity
2. **Value Promise (0:15-0:45):** What they'll learn, why it matters
3. **Context/Problem (0:45-2:00):** Establish need, relate to audience
4. **Core Teaching (2:00-7:00):** Break into 2-3 key concepts with examples
5. **Demo/Practice (7:00-9:30):** Hands-on application
6. **Summary/CTA (9:30-10:00):** Recap, next steps, call to action

**Actions:**
1. Create minute-by-minute outline with timing
2. Plan pattern interrupts (every 2-3 minutes)
3. Script the hook and value promise word-for-word
4. Identify where demos, examples, and practice occur
5. Design downloadable resource

**Output:** Detailed session outline with precise timing and content blocks

**Quality Check:** Can someone understand the value in the first 30 seconds?

---

### Step 5: Research-Specific Content
**Goal:** Validate your approach for this specific session/topic

**Actions:**
1. Search for best practices in this specific domain
2. Find proven frameworks or methodologies
3. Identify common pitfalls and how to avoid them
4. Discover compelling examples and case studies

**Output:** Topic-specific research findings

**Example:** For a prompt engineering session, research:
- Most effective prompt frameworks (CRAFT, RISEN, etc.)
- OpenAI's official best practices
- What specific techniques get best results

---

### Step 6: Full Script Development
**Goal:** Create word-for-word script with production notes

**Components:**
- Exact dialogue (what you'll say)
- Visual cues (what's on screen)
- Editing notes (transitions, effects)
- Timing markers (every 30-60 seconds)
- Pattern interrupts (music changes, visual shifts)

**Actions:**
1. Expand outline into full script
2. Write for speaking, not reading (conversational tone)
3. Include pause points for emphasis
4. Add production notes for editor
5. Create accessibility notes (captions, visuals for audio content)

**Output:** Complete script document ready for recording

**Writing Tips:**
- Read script aloud—if it sounds unnatural, rewrite
- Vary sentence length (short for emphasis, longer for explanation)
- Use "you" language (direct address)
- Include rhetorical questions to maintain engagement

---

### Step 7: Supporting Materials
**Goal:** Create all deliverables that enhance the core content

**Materials to Create:**
1. **Slide deck outline** - Key visuals for video
2. **Downloadable resources** - Templates, checklists, guides
3. **Practice exercises** - Hands-on activities
4. **Supplementary content** - Reading lists, tool links, community resources

**Actions:**
1. Design slide deck with consistent branding
2. Create high-value downloadable (25 prompt templates, etc.)
3. Develop practice exercises with clear instructions
4. Compile resource lists

**Output:** Complete supporting materials package

**Value Principle:** Downloadable resource alone should be worth the price of admission.

---

### Step 8: Production Workflow Planning
**Goal:** Determine fastest, most viable way to produce content

**Tiers to Consider:**
- **Tier 1: Maximum Speed** - Record once, light editing (2-3 hours)
- **Tier 2: Balanced** - Prepared outline, targeted editing (4-6 hours)
- **Tier 3: High Polish** - Full production (8-12 hours)

**Actions:**
1. Choose appropriate tier based on goals and constraints
2. Identify tools needed (recording, editing, graphics)
3. Create production checklist
4. Plan batch production if creating multiple videos

**Output:** Production workflow document with tool recommendations

**Key Decision:** Start with Tier 1, improve systematically.

## Applying This Process

### For a Single Session (10-min video):
**Total Time: 4-8 hours** (first time), 2-4 hours (optimized)

1. Structure (30 min) - Define session goals and flow
2. Research (60 min) - Find what works for this format/topic
3. Audience (30 min) - Quick persona and needs check
4. Outline (60 min) - Detailed timing and content blocks
5. Content Research (30 min) - Specific to this topic
6. Script (90 min) - Full word-for-word with production notes
7. Materials (60 min) - Slides and downloadable resource
8. Production (varies) - Record and edit based on tier chosen

### For a Complete Course (8 modules):
**Weeks 1-2:** Structure + Research + Audience (entire course)
**Weeks 3-10:** Create one module per week following steps 4-8
**Week 11-12:** Review, revise, package

## Key Decision Points

### When to Deep Dive vs. Move Fast?

**Deep Dive (More Research) When:**
- New topic area you're unfamiliar with
- High stakes (course launch, paying clients)
- Complex subject requiring precision
- Audience is skeptical or has specific objections

**Move Fast (Less Research) When:**
- Topic you know well
- Testing content viability
- Building momentum matters
- You can iterate based on feedback

### Content Depth: How Much is Enough?

**Go Deeper When:**
- Core skill that unlocks everything else
- Common point of failure for learners
- Unique value proposition / differentiator

**Stay High-Level When:**
- Context-setting or background
- Well-covered elsewhere (link to resources)
- Not critical path to main outcome

## Quality Checkpoints

### Before Publishing Any Training Content:

**Audience Fit:**
- [ ] Speaks to specific pain points
- [ ] Addresses common objections
- [ ] Appropriate level for target skill level
- [ ] Emotional tone matches audience state

**Structural Integrity:**
- [ ] Clear learning objective stated upfront
- [ ] Logical flow from point A to point B
- [ ] Pattern interrupts every 2-3 minutes
- [ ] Strong hook in first 15 seconds
- [ ] Clear call-to-action at end

**Practical Value:**
- [ ] At least one actionable takeaway
- [ ] Real examples, not hypotheticals
- [ ] Downloadable resource provides standalone value
- [ ] Participants can implement immediately

**Production Quality:**
- [ ] Audio is clear and professional
- [ ] Visuals support (not distract from) content
- [ ] Captions are accurate
- [ ] Accessible to target audience

## Common Pitfalls to Avoid

❌ **Curse of Knowledge:** Assuming audience knows basics
✅ **Beginner's Mind:** Explain every step, use analogies

❌ **Feature Dumping:** Teaching everything about a topic
✅ **Outcome Focus:** Teaching just what's needed for specific result

❌ **Theory First:** Long explanations before showing value
✅ **Value First:** Show the result, then explain how it works

❌ **Perfection Paralysis:** Waiting to ship until it's perfect
✅ **Iteration Mindset:** Ship good enough, improve based on feedback

❌ **One-Size-Fits-All:** Same content for all audience segments
✅ **Segmented Approach:** Customize for different personas

## Workflow Optimization Tips

### Batch Processing:
- Research all sessions at once
- Write all outlines in one sitting
- Record multiple videos in one day
- Edit similar tasks together

### Reusable Assets:
- Create intro/outro once, reuse
- Build slide templates
- Develop on-screen text library
- Save successful hooks/CTAs

### AI Assistance:
- Use ChatGPT/Claude for outline expansion
- Generate slide text variations
- Create description and SEO keywords
- Brainstorm examples and analogies

### Continuous Improvement:
- Track retention rates per video
- Note which sections get most comments/questions
- A/B test hooks and thumbnails
- Survey completers for feedback

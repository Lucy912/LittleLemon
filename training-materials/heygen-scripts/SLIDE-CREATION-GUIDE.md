# HeyGen Slide Creation Guide
## Best Practices for Creating Training Video Slides

**Created:** November 3, 2025
**Based on:** HeyGen 2025 specifications and video production best practices
**Purpose:** Guide for creating PowerPoint/PDF slides optimized for HeyGen video production

## üéØ Executive Summary

**Key Takeaways:**
- HeyGen supports up to **50 slides per video** (expanded from 15 in July 2025)
- **Two template options:** Editable Template (modifiable) vs Image Background (preserves design)
- **Recommended format:** PowerPoint (.pptx) or PDF
- **Optimal aspect ratio:** 16:9 (widescreen) for training content
- **Best tools:** PowerPoint, Canva, or Figma Slides
- **Design philosophy:** Simple, clear, video-optimized layouts

## üìä HeyGen Technical Specifications

### File Format Requirements

**Supported Formats:**
- ‚úÖ PowerPoint (.ppt, .pptx)
- ‚úÖ PDF files

**Slide Limits:**
- **Editable Template:** Maximum 50 slides per video
- **Image Background:** 50 slides recommended (no stated limit but practical maximum)

**File Size:** No specific limit mentioned, but keep reasonable for upload speed

---

### Template Options Explained

#### Option 1: Editable Template (Recommended for Most Cases)
**What it does:**
- Converts slides into modifiable content blocks
- Allows text changes and element repositioning in HeyGen
- Automatically extracts text for script generation

**Best for:**
- Standard presentations with text and images
- When you need flexibility to adjust content in HeyGen
- When using simple, clean designs

**Limitations:**
- Formatting may slightly differ from original
- Complex layouts might not translate perfectly
- Advanced animations/transitions may be lost

#### Option 2: Image Background (Recommended for Complex Designs)
**What it does:**
- Preserves original slide design as static backgrounds
- Maintains exact visual appearance
- Ideal for heavily designed presentations

**Best for:**
- Highly designed slides with complex layouts
- Custom graphics and illustrations
- Brand-specific designs that must remain exact
- Slides with special effects or gradients

**Limitations:**
- Text is not editable in HeyGen
- Less flexible for post-upload changes
- Requires speaker notes or script import for narration

## üé® Design Specifications for Training Videos

### Aspect Ratio & Resolution

**Recommended for Training Content:**
- **Aspect Ratio:** 16:9 (widescreen) - standard for YouTube, LMS platforms, and most training delivery
- **Resolution:** 1920 x 1080 pixels (1080p Full HD)
- **Alternative:** 1280 x 720 pixels (720p HD) if file size is concern

**PowerPoint Setup:**
1. Go to Design > Slide Size > Custom Slide Size
2. Select "Widescreen (16:9)" or "On-screen Show (16:9)"
3. Confirm dimensions: 13.333" x 7.5" (PowerPoint default for 16:9)

**Why 16:9?**
- Standard for most digital platforms (YouTube, Vimeo, LMS)
- Optimal for desktop and laptop viewing
- HeyGen supports multiple ratios, but 16:9 is most versatile for training
- Matches most learner viewing contexts

---

### Typography Guidelines

**Font Size Requirements:**

**For Standard Presentations (Editable Template):**
- **Minimum body text:** 18pt
- **Recommended body text:** 24-28pt
- **Headings:** 32-48pt
- **Titles:** 48-72pt

**For HD Video Production (Image Background):**
- **Minimum body text:** 48pt (critical for video clarity)
- **Recommended body text:** 54-60pt
- **Headings:** 72-96pt
- **Titles:** 96-120pt

**Font Choices:**
- **Safe for HeyGen:** Arial, Helvetica, Calibri, Open Sans, Roboto, Lato
- **Avoid:** Decorative fonts, script fonts, thin font weights
- **Best practice:** Use maximum 2 font families per presentation
- **Weight:** Use bold for emphasis, avoid light/thin weights for video

**Readability Tips:**
- High contrast: Dark text on light background or vice versa
- Avoid busy backgrounds behind text
- Test readability at actual viewing size
- Remember: Video compression can affect legibility

---

### Content Layout Best Practices

**Slide Density:**
- ‚úÖ **Maximum 4 bullet points per slide**
- ‚úÖ **Maximum 6-8 words per bullet**
- ‚úÖ **One main concept per slide**
- ‚ùå Avoid paragraph text on slides
- ‚ùå Avoid cramming multiple concepts

**White Space:**
- Leave 20-30% of slide empty
- Use generous margins (at least 0.5" from edges)
- Allow breathing room around elements
- Remember: Avatar will overlay part of the slide

**Avatar Space Consideration:**
- **Standard avatar position:** Lower right corner
- **Reserve area:** Bottom 25% of slide for avatar
- **Keep important content:** In top 75% of slide
- **Test placement:** Ensure avatar doesn't block key information

---

### Visual Elements

**Images:**
- **Resolution:** Minimum 300 DPI for print, 72-150 DPI for web/video
- **Size:** Use high-quality images that won't pixelate
- **Format:** PNG for graphics/logos (transparency), JPG for photos
- **Placement:** Leave space for avatar (avoid placing critical images bottom-right)

**Icons & Graphics:**
- Use simple, clear icons
- Maintain consistent style throughout
- Ensure sufficient contrast with background
- Use vector graphics when possible (SVG, or high-res PNG)

**Color Palette:**
- **Limit to 3-4 colors** per presentation
- Ensure accessibility: 4.5:1 contrast ratio minimum (WCAG AA standard)
- Test colors on different screens
- Avoid pure red/green combinations (colorblind consideration)

**Animations & Transitions:**
- **Use sparingly** - HeyGen may not preserve all animations
- Stick to simple transitions (fade, push) if using Editable Template
- For complex animations, use Image Background option
- Consider: May be better to add transitions in HeyGen editor

---

### Content Structure Per Slide

**Title Slides:**
- Video/module title
- Optional: Subtitle or brief description
- Keep clean and uncluttered
- Reserve bottom-right for avatar

**Content Slides:**
- Clear headline (what this slide is about)
- 2-4 bullet points OR one image with caption
- Minimal text - elaborate verbally in script
- Visual support for what's being said

**Example Slides:**
- Show before/after comparisons
- Use screenshots with annotations
- Highlight key areas with boxes/arrows
- Keep examples clear and uncluttered

**Summary Slides:**
- Bullet list of key takeaways (3-5 items)
- Visual checkmarks or icons
- Reinforce main points
- Preview next steps

## üìã Step-by-Step Workflow

### Phase 1: Planning (Before Creating Slides)

**Step 1: Review Your Script**
- Open your HeyGen script (e.g., `module-1.1-HEYGEN-SCRIPT.md`)
- Note the number of scenes (this = number of slides)
- Identify key visual moments in each scene

**Step 2: Gather Assets**
- Collect all images needed
- Prepare icons and graphics
- Ensure brand assets (logo, colors, fonts)
- Create any screenshots or diagrams

**Step 3: Create Asset Library**
```
/assets/
  /module-1/
    /images/
    /icons/
    /screenshots/
  /brand/
    logo.png
    color-palette.png
    fonts.txt
```

---

### Phase 2: Slide Creation

**Step 1: Set Up Master Template (Do Once)**

**In PowerPoint:**
1. Create new presentation
2. Set to 16:9 aspect ratio (Design > Slide Size)
3. View > Slide Master
4. Design master layout with:
   - Logo placement (top-left or bottom-center)
   - Font styles for titles and body
   - Color scheme
   - Background style
5. Save as template (.potx)

**In Canva:**
1. Create "Presentation (16:9)"
2. Set up brand kit (Pro feature) or save brand colors
3. Create 2-3 master layouts:
   - Title slide
   - Content slide (with bullet points)
   - Image slide
4. Save as template or duplicate for each video

**Step 2: Create Slides for First Video (Module 1.1)**

**Scene-by-Scene Approach:**
- One slide per scene in your script
- Follow slide specifications in script
- Keep text minimal (script provides narration)
- Add speaker notes with full script text

**Module 1.1 Example (10 slides needed):**
1. **Slide 1:** Split screen "45 min ‚Üí 4 min" transformation
2. **Slide 2:** "What You'll Learn Today" - 3 bullet points
3. **Slide 3:** Bad prompt example with generic output
4. **Slide 4:** Framework overlay on email example
5. **Slide 5:** Table showing data analysis output
6. **Slide 6:** LinkedIn post with framework callouts
7. **Slide 7:** Large 5-part framework graphic
8. **Slide 8:** Template library previews (8 categories)
9. **Slide 9:** Checklist of what was covered
10. **Slide 10:** CTA buttons (Subscribe, Download, Comment)

**Step 3: Add Speaker Notes (Critical!)**
- Copy script text for each scene into PowerPoint speaker notes
- HeyGen can extract these automatically
- Ensures sync between slides and narration
- Alternative: Upload script separately in HeyGen

---

### Phase 3: Quality Check

**Before Exporting:**
- [ ] All slides in 16:9 aspect ratio
- [ ] Font sizes meet minimum requirements (18pt or 48pt depending on method)
- [ ] High contrast for readability
- [ ] Images are high quality (not pixelated)
- [ ] Important content in top 75% (avatar space reserved)
- [ ] Consistent branding throughout
- [ ] Speaker notes added (if using)
- [ ] Total slides ‚â§ 50
- [ ] Spelling and grammar checked
- [ ] Visual hierarchy clear

**Test Slide:**
- View at 50% zoom - still readable?
- Check on phone screen if possible
- Ask: Would this work with an avatar overlayed?

---

### Phase 4: Export for HeyGen

**PowerPoint Export:**
- Save as .pptx (not .ppt)
- File > Save As > PowerPoint Presentation (.pptx)
- Keep file size reasonable (<50MB if possible)

**Canva Export:**
- File > Download > Microsoft PowerPoint
- Or: Download > PDF (if using Image Background method)
- Verify export looks correct

**Figma Export:**
- Export as PDF (File > Export)
- High quality settings
- Use Image Background option in HeyGen

**Google Slides Export:**
- File > Download > Microsoft PowerPoint (.pptx)
- Test exported file opens correctly

## üé¨ HeyGen Upload & Configuration

### Uploading to HeyGen

**Step 1: Import Presentation**
1. Log into HeyGen AI Studio
2. Click "Create" or "New Video"
3. Select "Upload Presentation"
4. Choose your .pptx or PDF file
5. Wait for processing (1-3 minutes)

**Step 2: Choose Template Option**

**Choose Editable Template if:**
- ‚úÖ Using simple, clean slide designs
- ‚úÖ Want to adjust text in HeyGen
- ‚úÖ Need flexibility for changes
- ‚úÖ Using speaker notes for script
- ‚úÖ Standard fonts and layouts

**Choose Image Background if:**
- ‚úÖ Highly designed slides
- ‚úÖ Complex layouts or graphics
- ‚úÖ Brand-specific designs that must be exact
- ‚úÖ Using gradients or special effects
- ‚úÖ Will import script separately

**Step 3: Script Assignment**

**Option A: Extract from Speaker Notes**
- HeyGen automatically extracts speaker notes
- Assign to corresponding slides
- Review and adjust timing

**Option B: Manual Script Import**
- Copy script from your .md file
- Paste into HeyGen script editor
- Assign sections to slides manually
- Adjust timing per slide

**Step 4: Avatar Selection**
- Choose avatar appropriate for content
- Position avatar (typically lower right)
- Test that avatar doesn't block content

**Step 5: Voice & Settings**
- Select voice (if not using avatar's default)
- Adjust speaking speed if needed
- Set language (English for these scripts)
- Preview first few slides

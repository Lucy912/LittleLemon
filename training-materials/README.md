## How to Use These Materials

### For Session 1 Production:

**Week 1: Pre-Production**
1. Read session-1-full-script.md
2. Practice delivery (don't memorize, internalize flow)
3. Set up recording software (OBS Studio or ScreenFlow)
4. Prepare ChatGPT examples (copy-paste ready)
5. Create 5-10 key slides from session-1-slide-outline.md

**Week 2: Production**
1. Record video following script (2-3 hours)
2. Edit following production checklist (2-3 hours)
3. Create thumbnail (3 versions for A/B test)
4. Export 25-prompt template library as PDF

**Week 3: Publishing**
1. Upload to YouTube
2. Add captions (auto-generate, then edit)
3. Write description with timestamps
4. Set up end screen
5. Publish and promote

### For Future Sessions:

1. Follow the Training Content Creator Skill methodology
2. Reference Session 1 as a template
3. Adapt based on topic and audience
4. Maintain consistent quality standards

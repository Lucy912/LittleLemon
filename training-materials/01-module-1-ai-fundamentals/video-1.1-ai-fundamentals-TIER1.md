## Production Checklist

### PREP (30 minutes)
- [ ] Review this outline (don't memorize, internalize flow)
- [ ] Prepare 3 ChatGPT examples (copy-paste ready)
- [ ] Clean desktop, close unnecessary tabs
- [ ] Test audio (30-second recording)
- [ ] Have 25-prompt template open for demo

### RECORD (60-90 minutes)
- [ ] Record in ONE TAKE (don't stop for small mistakes)
- [ ] If you mess up badly: pause 3 seconds, restart sentence
- [ ] Keep energy high, speak clearly
- [ ] Total recording: 12-15 min (you'll cut to 10)

### EDIT (45-60 minutes)
- [ ] Cut out long pauses (over 2-3 seconds)
- [ ] Remove major mistakes only
- [ ] Add 3-5 text overlays for key points
- [ ] Add simple title + end screen
- [ ] Export 1080p, 30fps

### PUBLISH (15-30 minutes)
- [ ] Upload to YouTube
- [ ] Auto-captions (quick edit)
- [ ] Description + timestamps
- [ ] Simple thumbnail (Canva template, 10 min)
- [ ] SHIP IT

**Total: 2.5-3.5 hours**
